# AVA Markets

AVA Markets is a premium market intelligence platform for crypto, stocks, and commodities.

It combines:
- live-style market snapshots
- AVA Brain signal logic
- trend analysis
- forecast layers
- premium account access
- Stripe-powered subscriptions
- admin and dashboard tools

## Features

- Top 100 crypto market tracking
- Global stocks and commodities coverage
- Live-style candle visualization
- BUY / SELL / HOLD signal layers
- Trend intelligence
- Forecast insights
- User registration and login
- Premium subscription plans
- Stripe checkout and billing portal
- Admin dashboard
- Snapshot caching for faster performance
